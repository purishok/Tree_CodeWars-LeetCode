# Binary Tree Algorithms
Колекція алгоритмів для роботи з бінарними деревами, реалізованих на Python.
Вміст

# Tree Traversals
Три класичні методи обходу дерева в глибину:

Pre-order — Вузол → Ліво → Право
In-order — Ліво → Вузол → Право. Для BST повертає елементи у відсортованому порядку
Post-order — Ліво → Право → Вузол. Корисний для видалення дерева або обчислення виразів


# Level-order Traversal
Обхід дерева за рівнями зліва направо. Корінь іде першим, потім його діти, і так далі. Використовує рекурсивний підхід із словником для групування вузлів за рівнями.

# BST Node Deletion
Видалення вузла з бінарного дерева пошуку за заданим ключем. Розглядає три випадки: вузол є листком, має одного дочірнього, або має двох дочірніх — у останньому випадку замінює значення інордер-наступником.
